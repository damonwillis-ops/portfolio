APERTURE — AI Camera Design Assurance
Production

Enterprise physical security programs spend millions installing cameras that fail in the field — wrong angles, coverage gaps, AI-CV model mismatches. APERTURE moves defect detection upstream. It ingests Axis Site Designer exports (WGS84 geocoordinate precision), enriches camera placements with canonical store-space ground truth (departments, fixtures, thermal zones), and validates every placement against codified AI-CV deployment requirements before a truck ever rolls.

Automated design review gating install sign-off across thousands of store locations.
Eliminated manual data entry across hundreds of thousands of device records.
FATHOM — 20-Agent Intelligence Orchestration Engine
Production

CCTV program data across thousands of locations goes stale fast. FATHOM is a 20-agent orchestration system that keeps program data fresh, risk-scored, and insight-ready. Agents run independently on strict latency SLOs; results surface through a dual-view dashboard (executive summary + operator depth). Built on FastAPI + React + SQLite.

Program leadership gets a live operational cockpit instead of weekly manual reports.
HORIZON — 6-Layer Program Decision System
Production

Physical security programs make decisions on bad data. HORIZON is a layered decision system built on top of the program's QuickBase data, moving from raw append-only mirror through canonical keys, semantic views, versioned metrics, and agent I/O — to produce traceable, definition-locked answers to business questions that previously required four analysts and a week.

Single source of truth for CCTV program KPIs with full lineage and SLO enforcement.
