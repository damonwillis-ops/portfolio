Systems Built

APERTURE — AI Camera Design Assurance
Production

Camera placement defects were discovered after installation — truck rolls, rework, and cost. APERTURE validates every placement against computer vision deployment requirements before installation. It ingests Axis Site Designer exports with WGS84 geocoordinate precision, enriches placements with canonical store-space ground truth, and gates install sign-off with requirement-traceable design reviews.

Defects become findings before they become field problems — at 4,600-location scale.
Manual device data entry eliminated across hundreds of thousands of records.
FATHOM — 20-Agent Intelligence Orchestration Engine
Production

Program decisions relied on stale, scattered data with no single source of truth. FATHOM is a 20-agent orchestration system that keeps enterprise security program data continuously current, risk-scored, and ready. Agents run independently on strict latency SLOs; results surface through a dual-view dashboard covering both executive and operator needs. Built on FastAPI + React + SQLite.

Leadership gets a live operational picture instead of weekly manual reports.
HORIZON — 6-Layer Program Decision System
Production

Leadership questions required manual data pulls — slow, inconsistent, and impossible to trace. HORIZON answers enterprise security program questions with traceable, consistent data under strict response time standards. Built in six layers from raw data through canonical keys, semantic views, versioned metrics, and agent I/O. Every answer carries a single trace ID.

Single source of truth for program KPIs with full lineage and SLO enforcement.
