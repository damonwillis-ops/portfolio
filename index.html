Systems Built

What I've built.

Not proof-of-concept tools. Each one runs against real program data at enterprise scale, with real operational stakes.

APERTURE — AI Camera Design Assurance
Production

Enterprise physical security programs routinely sign off on camera designs without validating AI-CV deployment requirements — gaps surface post-install, driving costly truck rolls and rework. APERTURE moves sign-off upstream: ingests Axis Site Designer exports with WGS84 geocoordinate precision, enriches placements with canonical store-space ground truth, and produces requirement-traceable design reviews that gate installation.

Defects become findings before they become field problems — at 4,600-location scale.
Device data entry automated across hundreds of thousands of records.
FATHOM — 20-Agent Intelligence Orchestration Engine
Production

At enterprise scale, security program data degrades faster than any team can manually refresh it. FATHOM is a 20-agent orchestration system that keeps program data continuously current, risk-scored, and ready without manual intervention. Agents run independently on strict latency SLOs; results surface through a dual-view dashboard covering executive and operator needs. Built on FastAPI + React + SQLite.

Leadership gets a live operational picture — continuously current, no manual refresh cycle.
HORIZON — 6-Layer Program Decision System
Production

Enterprise security programs lack a consistent, traceable way to answer leadership questions about program health and risk posture. HORIZON is a six-layer decision system built to close that gap — moving from raw data through canonical keys, semantic views, versioned metrics, and agent I/O to deliver answers under strict response time standards. Every answer carries a single trace ID.

Single source of truth for program KPIs with full lineage and SLO enforcement.
